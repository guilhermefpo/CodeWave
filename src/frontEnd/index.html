Gráficos Demográficos de SJC

Um projeto de criação de uma plataforma para filtrar e exibir dados demográficos e estatísticos de São José dos Campos, permitindo buscas por bairro, período e indicadores específicos. A interface apresentará as informações de forma clara, com gráficos interativos e tabelas para facilitar a análise.
